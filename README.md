# Sistema de Análise de Risco Financeiro

> Projeto de Conclusão de Curso — Hackathon Ada Elas + Tech 
> Tema 3 – Risco Financeiro (Risk Intelligence)

---

## Sobre o projeto

Aplicação web que analisa o perfil financeiro de um usuário e identifica riscos automaticamente. O sistema processa transações mensais, calcula um score de risco de 0 a 100, detecta gastos fora do padrão histórico e gera alertas em tempo real.

---

## Tecnologias
- React
- Vite
- TypeScript

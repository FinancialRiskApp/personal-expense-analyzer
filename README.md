# Personal Expense Analyzer

> Projeto de Conclusão de Curso — Hackathon Ada Elas + Tech 
> Tema 3 – Risco Financeiro (Risk Intelligence)

---

## Sobre o projeto

O Personal Expense Analyzer é uma aplicação web para análise de risco financeiro pessoal. O usuário informa sua renda mensal e cadastra seus gastos por mês, categoria, descrição e valor. A partir desses dados, o sistema calcula saldo projetado, identifica gastos fora do padrão, gera alertas de risco e apresenta um score financeiro simples para apoiar a tomada de decisão.

---

## Tecnologias Utilizadas

- React;
- TypeScript;
- Tailwind CSS;
- JavaScript;
- JSON para simulação inicial de dados;
- LocalStorage para persistência simples dos dados cadastrados.

## Funcionalidades

- Cadastro da renda mensal do usuário;
- Cadastro de gastos por mês, descrição, valor e categoria;
- Listagem dos gastos em tabela;
- Filtro de gastos por mês;
- Cálculo de saldo projetado;
- Detecção de gastos fora do padrão;
- Geração de alertas de risco financeiro;
- Cálculo de score financeiro simples;
- Visualização de distribuição dos gastos por categoria.

## Fluxo de Uso

1. O usuário informa sua renda mensal.
2. O usuário cadastra seus gastos, indicando mês, descrição, valor e categoria.
3. A aplicação organiza os gastos em uma tabela.
4. O usuário pode filtrar os gastos por mês.
5. O dashboard calcula automaticamente os indicadores financeiros com base nos dados cadastrados.

## Proposta de Valor

A proposta do Personal Expense Analyzer é transformar registros simples de gastos em uma leitura rápida de risco financeiro. Em vez de apenas listar despesas, a aplicação interpreta os dados e apresenta sinais de alerta, permitindo que o usuário perceba quando seus gastos estão comprometendo sua renda ou fugindo do padrão esperado.
